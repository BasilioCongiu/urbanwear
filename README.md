UrbanWear - Landing Page

Landing page dinamica progettata per un brand di abbigliamento. L'obiettivo principale è stato creare un'esperienza utente (UX) fluida e ottimizzata per dispositivi mobili.

Tecnologie Utilizzate
- **HTML5 & CSS3**: Struttura semantica e layout responsive.
- **JavaScript (ES6+)**: Logica per le interazioni principali dell'interfaccia.
- **Mobile First Approach**: Design ottimizzato per garantire una navigazione fluida su smartphone e desktop.

Cosa ho imparato
- Gestione della gerarchia visiva tramite CSS Grid e Flexbox.
- Implementazione di componenti interattivi per migliorare il tasso di conversione dell'utente.
- Ottimizzazione dei tempi di caricamento delle immagini e degli asset.
